# 5大Tab UI设计

> 来源：PRD v2.0 第4章

## Tab1 首页

**定位**：用户每日打开App的第一屏，传递关怀感和训练节奏感。

**功能列表：**

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 个性化问候 | 根据时间段（早/午/晚）和用户训练历史生成问候语 | P0 |
| 教练在线状态 | 显示当前选择的AI教练"在线"，带有呼吸灯动效 | P0 |
| 签到入口 | 今日签到按钮，已签到显示"已签到"+连续天数 | P0 |
| 今日推荐 | 基于用户能力短板推荐1个场景训练，附推荐理由 | P1 |
| 最近训练 | 展示最近1-3条训练记录（场景名+得分+时间） | P1 |
| 本周进度 | 本周训练次数/目标次数，环形进度条 | P1 |
| 晚安计划 | 如果已设置，显示今晚计划时间和教练；如果未设置，引导设置 | P2 |
| 变美联动 | 与"悦己颜值社"联动卡片，展示形象提升建议 | P2 |

**首页布局（从上到下）：**

```
┌─────────────────────────────┐
│  个性化问候 + 教练在线状态     │
├─────────────────────────────┤
│  签到入口（大按钮）           │
├─────────────────────────────┤
│  今日推荐场景卡片             │
├─────────────────────────────┤
│  最近训练记录                 │
├─────────────────────────────┤
│  本周训练进度                 │
├─────────────────────────────┤
│  晚安计划 / 变美联动          │
└─────────────────────────────┘
```

## Tab2 教练页

**定位**：AI教练的核心展示与交互页面，用户与教练的"面对面"空间。

**功能列表：**

| 功能 | 描述 | 优先级 |
|------|------|--------|
| AI教练全屏展示 | 教练立绘占屏幕70%高度，底部为交互区域 | P0 |
| 训练积分进度条 | 顶部显示当前训练积分和距离下一等级的进度 | P0 |
| 对话训练入口 | "开始对话训练"按钮，进入与教练的自由对话模式 | P0 |
| 场景挑战入口 | "场景挑战"按钮，跳转到场景页选择具体场景 | P0 |
| 朋友圈 | 教练朋友圈入口，显示教练最新一条动态 | P1 |
| 今日任务 | 展示今日推荐的训练任务（1-3个），完成可获得额外积分 | P1 |

**教练页布局：**

```
┌─────────────────────────────┐
│  训练积分进度条 + 学员等级(如:黄金学员)│
├─────────────────────────────┤
│                             │
│    AI教练全屏立绘 (70%)       │
│    (Spine/Live2D/3D渲染)     │
│                             │
├─────────────────────────────┤
│  [开始对话训练] [场景挑战]     │
├─────────────────────────────┤
│  教练朋友圈(最新1条)          │
├─────────────────────────────┤
│  今日任务列表                 │
└─────────────────────────────┘
```

**教练渲染交互规格：**

- **待机状态**：教练轻微呼吸动画 + 偶尔眨眼（3-5秒一次）
- **对话中**：教练表情随对话内容变化，口型同步TTS语音
- **反馈时**：点头认可（正面反馈）/ 眼神鼓励（建设性反馈）/ 举手击掌（鼓励与庆祝）
- **点击教练**：触发随机互动动画（眨眼/歪头/挥手）

## Tab3 场景页

**定位**：社交模拟训练的场景选择中心，按社交阶段组织和逐步解锁。

**功能列表：**

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 社交阶段分组 | 按六大社交阶段分组展示场景 | P0 |
| 场景卡片 | 每个场景显示：名称、难度（星级）、预估时长、完成度 | P0 |
| 阶段逐步解锁 | 需满足前置条件才能解锁下一阶段 | P0 |
| 训练评估报告 | 每次训练结束后生成评估报告 | P0 |
| 高难度关卡 | 训练积分≥300后解锁的额外高难度场景 | P1 |

**场景卡片信息：**

```
┌─────────────────────────┐
│  💑 相亲模拟             │
│  ⭐⭐ 中等  ·  约12分钟   │
│  ━━━━━━━━░░  75%       │
│  [开始训练]              │
└─────────────────────────┘
```

**卡片字段说明：**

- **场景名称**：直观描述社交场景
- **难度**：1-5星，用⭐图标表示
- **预估时长**：以分钟为单位
- **完成度**：0-100%，基于最佳成绩计算
- **开始训练按钮**：未解锁时灰色+解锁条件提示

**阶段解锁规则详见第6章。**

## Tab4 成长页

**定位**：用户的社交能力成长数据中心，可视化呈现进步轨迹。

**功能列表：**

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 能力雷达图 | 五维能力：沟通力/表达力/共情力/情绪控制/应变力 | P0 |
| 进步曲线 | 各维度能力随时间的变化曲线 | P0 |
| 情绪日记 | 训练后自动生成 + 手动补充 | P1 |
| 里程碑 | 重要成就节点展示 | P1 |
| 学习卡片 | 收藏的学习卡片列表 | P1 |
| 每周报告 | 每周一自动生成，含训练统计+能力变化+教练寄语 | P2 |
| 教练寄语 | AI教练针对用户近期的个性化鼓励 | P2 |

**能力雷达图五维度：**

| 维度 | 说明 | 计算来源 |
|------|------|---------|
| 沟通力 | 信息传达的清晰度和有效性 | 对话任务完成度+表达评分 |
| 表达力 | 语言的丰富度和感染力 | 词汇多样性+情感表达评分 |
| 共情力 | 理解他人感受和需求的能力 | 共情类场景评分+NVC练习 |
| 情绪控制 | 在压力场景中保持冷静的能力 | 高难度场景评分+情绪日记 |
| 应变力 | 面对意外情况的灵活反应能力 | 突发分支场景评分+随机事件 |

**每维度初始值50分，满分100分，根据训练表现动态调整。**

## Tab5 我的页

**定位**：用户个人中心和设置入口。

**功能列表：**

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 用户信息 | 头像、昵称、学员等级、训练积分 | P0 |
| 形象展示 | 用户选择的个人形象/头像框 | P1 |
| 教练设置 | 当前教练选择、自定义教练入口 | P0 |
| 会员中心 | 会员方案展示、购买、续费 | P0 |
| 签到 | 每日签到入口（与首页同步） | P0 |
| 学习卡片 | 已收藏的学习卡片管理 | P1 |
| 变美联动 | 与"悦己颜值社"的联动设置 | P2 |
| 隐私设置 | 数据共享、隐私政策 | P0 |
| 实名认证 | 实名认证状态和管理 | P0 |
| 防沉迷 | 训练时长设置、休息提醒 | P0 |
| 重置训练 | 清除训练数据，重新开始（需二次确认） | P2 |
