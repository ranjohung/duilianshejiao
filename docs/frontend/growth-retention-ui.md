## 时空穿梭与学习卡片系统

### 时空穿梭

**功能描述：** 在训练对话中，用户可以使用"时空穿梭券"回到关键分支点，重新选择不同的对话选项。

**使用流程：**

1. 训练中，每轮选择后显示"穿梭"按钮
2. 点击"穿梭"→消耗1张时空穿梭券
3. 画面出现时空倒流动效
4. 回到该轮选择前，重新展示A/B/C选项
5. 用户重新选择，教练基于新选择继续对话
6. 原选择和新选择的对比展示（可选）

**限制规则：**

- 每次训练最多使用2张
- 只能回退到当前轮次，不能跨轮回退
- 使用穿梭券不影响训练评分（鼓励试错学习）
- 穿梭后教练会特别点评两个选择的差异

**时空穿梭券获取方式：**

| 获取渠道 | 数量 | 说明 |
|---------|------|------|
| 每日签到 | 第3、5、7天各获得1张 | 签到奖励 |
| 周卡会员 | 每周3张 | 会员权益 |
| 月卡会员 | 每月10张 | 会员权益 |
| 年卡会员 | 每月15张 | 会员权益 |
| 邀请好友 | 各获得1张 | 社交裂变 |

## 留存与增长功能

### 晚安计划

| 项目 | 规格 |
|------|------|
| 触发时间 | 每晚用户设定时间（默认21:30） |
| 内容 | AI教练语音陪伴入睡（3-5分钟轻声说话） |
| 内容类型 | 今日回顾(40%)+明日期待(30%)+舒缓引导(30%) |
| 设置 | 用户可自定义时间和开关。设置入口：Tab5我的页→晚安计划设置；Tab1首页→晚安计划卡片（显示当前设置状态，点击进入设置） |
| 生成 | node-cron定时任务触发，TTS生成语音 |

### 情绪日记

| 项目 | 规格 |
|------|------|
| 自动生成 | 每次训练结束后，基于训练表现和对话内容自动生成 |
| 手动补充 | 用户可编辑自动生成的日记，添加个人感受 |
| 日记内容 | 情绪类型（快乐/平静/紧张/焦虑/沮丧）、强度(1-10)、文字描述 |
| 查看方式 | 成长页→情绪日记列表，按时间倒序 |
| 数据分析 | 每周报告包含情绪变化趋势 |

### 真实挑战

| 项目 | 规格 |
|------|------|
| 定义 | 线下真实社交任务 |
| 示例 | "今天主动和一位同事聊天5分钟""在咖啡厅向陌生人搭话" |
| 完成方式 | 上传文字描述/图片凭证 |
| 审核方式 | AI教练自动验证（关键词+图片分析） |
| 奖励 | 20积分 + 对应才艺经验 |
| 频率 | 每日最多完成1个，不可重复完成 |

**真实挑战完成流程：**

1. 用户查看本周挑战列表（Tab4成长页→真实挑战）
2. 选择挑战任务，点击"开始挑战"
3. 用户在线下完成任务
4. 回到App，点击"打卡完成"
5. 用户填写文字描述（100字以上）+ 可选上传图片
6. AI教练自动验证（关键词+图片分析）
7. 验证通过 → 获得奖励（20积分+对应才艺经验）
8. 验证不通过 → AI给出建议，用户可重新提交
9. 每日最多完成1个真实挑战
10. 挑战不可重复完成
