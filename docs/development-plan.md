## 关键技术实现要点

### 1. 动态深度对话引擎
- 移除固定轮数限制，改为话题覆盖完成自动结束
- NPC回应基于LLM动态生成，不依赖预设剧本
- 参考选项作为辅助，用户可完全自由输入

### 2. 双向评分算法
```
score_delta = LLM分析得出的得分(-10~+10)
total_score = Math.max(-100, Math.min(100, total_score + score_delta))

if (score_delta > 0) {
    positiveScore += score_delta
    consecutiveNegativeRounds = 0
} else if (score_delta < 0) {
    negativeScore += Math.abs(score_delta)
    consecutiveNegativeRounds++
}

netScore = positiveScore - negativeScore
```

### 3. 情绪识别三层引擎
- 文本层（60%权重）：LLM分析用户输入情绪
- 语音层（30%权重）：Azure Speech分析语速/音调/音量
- 表情层（10%权重）：预留接口，MVP暂不实现

### 4. 场景解锁逻辑
```
if (用户是会员) {
    直接解锁所有场景
} else {
    检查净得分是否达标
    if (netScore >= 阶段要求) {
        解锁下一阶段场景
    }
}
```
