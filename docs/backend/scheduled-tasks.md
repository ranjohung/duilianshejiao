# 定时任务与事件驱动

> 来源：PRD v2.0 多章节中定时任务相关内容

## 技术选型

定时任务基于 **node-cron** 实现，集成于 Node.js + Express 后端服务中。

| 组件 | 技术选型 | 说明 |
|------|---------|------|
| 定时任务 | node-cron | 签到重置、每周报告生成、晚安计划触发、朋友圈自动发布 |

## 定时任务清单

### 1. 签到重置

- **触发规则**：每日 00:00
- **功能**：重置用户签到状态，更新连续签到天数
- **数据操作**：更新 check_ins 表相关状态

### 2. 晚安计划触发

| 项目 | 规格 |
|------|------|
| 触发时间 | 每晚用户设定时间（默认21:30） |
| 内容 | AI教练语音陪伴入睡（3-5分钟轻声说话） |
| 内容类型 | 今日回顾(40%)+明日期待(30%)+舒缓引导(30%) |
| 设置 | 用户可自定义时间和开关。设置入口：Tab5我的页→晚安计划设置；Tab1首页→晚安计划卡片（显示当前设置状态，点击进入设置） |
| 生成 | node-cron定时任务触发，TTS生成语音 |

**内部接口：** `POST /api/goodnight/trigger`

```json
{
  "user_id": 10001,
  "plan_id": 1
}
```

**响应：**

```json
{
  "code": 0,
  "data": {
    "audio_url": "https://oss.example.com/goodnight/10001_20260715.mp3",
    "duration_seconds": 180,
    "content_preview": "今天你完成了两次训练，进步很大..."
  }
}
```

### 3. 每周报告生成

| 项目 | 规格 |
|------|------|
| 生成时间 | 每周一 08:00 |
| 生成方式 | node-cron定时任务自动生成 |
| 报告内容 | 本周训练统计、各维度能力变化、情绪变化趋势、教练寄语 |
| 推送方式 | App内通知+报告卡片 |

### 4. 朋友圈自动发布

| 项目 | 规则 |
|------|------|
| 生成频率 | 每个教练每日1条 |
| 生成时间 | 随机在8:00-22:00之间 |
| 动态类型 | 生活分享(40%)、心情表达(30%)、训练鼓励(30%) |
| 评论功能 | 用户可评论，教练会回复（延迟5-30分钟模拟真实感） |

**技术实现：**

- 定时任务(node-cron)每日为每个教练生成1条朋友圈
- 朋友圈内容由LLM生成，基于教练人格+当前情绪+用户相关记忆
- 评论回复由LLM实时生成
- 朋友圈列表按时间倒序，保留最近30条

### 5. 训练次数重置

- **触发规则**：每周一 00:00
- **功能**：重置用户 `weekly_training_count`，更新 `weekly_count_reset_at`
- **数据操作**：更新 users 表的周训练次数字段

### 6. 每日训练时长重置

- **触发规则**：每日 00:00
- **功能**：重置用户 `daily_training_minutes`，更新 `daily_minutes_reset_at`
- **数据操作**：更新 users 表的日训练时长字段

### 7. 教练情绪每日重置

- **触发规则**：每日 00:00
- **功能**：将教练情绪状态重置为默认值（愉悦→70，焦虑→30，疲惫→20）
- **数据操作**：更新 coaches 表的 `emotion_state` JSON字段

### 8. 会员过期检查

- **触发规则**：每日 01:00
- **功能**：检查会员是否过期，过期则降级为免费用户
- **数据操作**：更新 memberships 表和 users 表的会员等级

## 任务调度汇总

| 任务 | cron表达式 | 说明 |
|------|-----------|------|
| 签到重置 | `0 0 * * *` | 每日00:00 |
| 每日训练时长重置 | `0 0 * * *` | 每日00:00 |
| 教练情绪重置 | `0 0 * * *` | 每日00:00 |
| 会员过期检查 | `0 1 * * *` | 每日01:00 |
| 朋友圈自动发布 | `0 8-22 * * *` | 每日8:00-22:00 |
| 晚安计划触发 | `30 21 * * *` | 每日21:30(默认) |
| 每周报告生成 | `0 8 * * 1` | 每周一08:00 |
| 训练次数周重置 | `0 0 * * 1` | 每周一00:00 |
