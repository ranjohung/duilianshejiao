# 留存与增长功能

> 来源：PRD v2.0 第17章

## 17. 留存与增长功能

### 17.1 晚安计划

| 项目 | 规格 |
|------|------|
| 触发时间 | 每晚用户设定时间（默认21:30） |
| 内容 | AI教练语音陪伴入睡（3-5分钟轻声说话） |
| 内容类型 | 今日回顾(40%)+明日期待(30%)+舒缓引导(30%) |
| 设置 | 用户可自定义时间和开关。设置入口：Tab5我的页→晚安计划设置；Tab1首页→晚安计划卡片（显示当前设置状态，点击进入设置） |
| 生成 | node-cron定时任务触发，TTS生成语音 |

### 17.2 情绪日记

| 项目 | 规格 |
|------|------|
| 自动生成 | 每次训练结束后，基于训练表现和对话内容自动生成 |
| 手动补充 | 用户可编辑自动生成的日记，添加个人感受 |
| 日记内容 | 情绪类型（快乐/平静/紧张/焦虑/沮丧）、强度(1-10)、文字描述 |
| 查看方式 | 成长页→情绪日记列表，按时间倒序 |
| 数据分析 | 每周报告包含情绪变化趋势 |

### 17.3 每周报告

| 项目 | 规格 |
|------|------|
| 生成时间 | 每周一 08:00 |
| 生成方式 | node-cron定时任务自动生成 |
| 报告内容 | 本周训练统计、各维度能力变化、情绪变化趋势、教练寄语 |
| 推送方式 | App内通知+报告卡片 |
| 保存期限 | 最近12周 |

  完成训练：5次
  训练时长：65分钟
  获得积分：85分

📈 能力变化
  沟通力：52 → 58 (+6) ⬆️

### 17.4 签到系统

| 项目 | 规格 |
|------|------|
| 签到时间 | 每日 00:00-23:59 |
| 签到奖励 | 基础5积分 + 连续天数递增(5,6,7,8,9,10,11) |
| 连续7天奖励 | 额外20积分 + 1张时空穿梭券 + 1张双倍积分卡 |
| 断签处理 | 连续天数归零，从5积分重新开始 |
| 补签 | 不可补签 |
| 签到入口 | 首页签到按钮 + 我的页签到入口 |

### 17.5 真实挑战

| 项目 | 规格 |
|------|------|
| 定义 | 线下真实社交任务 |
| 示例 | "今天主动和一位同事聊天5分钟""在咖啡厅向陌生人搭话" |
| 完成方式 | 上传文字描述/图片凭证 |
| 审核方式 | AI教练自动验证（关键词+图片分析） |
| 奖励 | 20积分 + 对应才艺经验 |
| 频率 | 每日最多完成1个，不可重复完成 |

**真实挑战完成流程：**

1. 用户查看本周挑战列表（Tab4成长页→真实挑战）
2. 选择挑战任务，点击"开始挑战"
3. 用户在线下完成任务
4. 回到App，点击"打卡完成"
5. 用户填写文字描述（100字以上）+ 可选上传图片
6. AI教练自动验证（关键词+图片分析）
7. 验证通过 → 获得奖励（20积分+对应才艺经验）
8. 验证不通过 → AI给出建议，用户可重新提交
9. 每日最多完成1个真实挑战
10. 挑战不可重复完成
