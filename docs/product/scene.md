# 社交模拟场景与高难度关卡

> 来源：PRD v2.0 第6-7章

## 6. 社交模拟场景系统

### 6.1 六大社交阶段

| 阶段 | 名称 | 核心能力 | 描述 |
|------|------|---------|------|
| 阶段一 | 破冰期 | 开启对话、第一印象 | 学习如何自然地开始一段对话 |
| 阶段二 | 接触期 | 维持对话、寻找共同话题 | 学习如何让对话持续下去 |
| 阶段三 | 熟悉期 | 深度交流、情感表达 | 学习如何表达真实的自己 |
| 阶段四 | 亲密期 | 冲突处理、深层共情 | 学习如何在矛盾中沟通 |
| 阶段五 | 职场期 | 职场沟通、专业表达 | 学习职场中的得体沟通 |
| 阶段六 | 高难期 | 压力沟通、谈判博弈 | 学习在高压场景中保持冷静 |

### 6.3 训练流程

```
开始训练
  │
  ▼
场景介绍（15秒）
  │  - 展示场景背景、对方角色、训练目标
  │
  ▼
对话轮次（核心环节，3-8轮）
  │  ┌─────────────────────────────────┐
  │  │  教练/AI角色说话                  │
  │  │         ↓                       │
  │  │  展示A/B/C三个选项               │
  │  │    A: 直接表达型                  │
  │  │    B: 委婉表达型                  │
  │  │    C: 回避/幽默型                 │
  │  │         ↓                       │
  │  │  用户选择 → 教练即时反馈           │
  │  │  （指出选择的优劣+改进建议）        │
  │  └─────────────────────────────────┘
  │
  ▼
评估报告
  │  - 总分(0-100)
  │  - 各维度得分
  │  - 星级评价(1-3星)
  │  - 优势亮点
  │  - 改进建议
  │  - 教练寄语
  │
  ▼
引导卡片
  │  - 本次训练的关键知识点
  │  - 可收藏、可分享
  │
  ▼
结束（返回场景页）
```

**选项设计规则：**

- A选项：直接/主动型——效果通常最积极，但需考虑场合
- B选项：委婉/间接型——社交安全但可能效果不足
- C选项：回避/幽默型——避免冲突但可能错过沟通机会
- 每轮选项设计必须体现不同的沟通策略，无绝对正确答案
- 选项效果根据场景上下文动态评估，不是固定得分

### 6.4 阶段解锁规则

| 阶段 | 解锁条件 |
|------|---------|
| 阶段一 | 默认解锁 |
| 阶段二 | 完成阶段一至少1个场景 + 训练积分 ≥ 50 |
| 阶段三 | 完成阶段二至少1个场景 + 训练积分 ≥ 100 |
| 阶段四 | 完成阶段三至少1个场景 + 训练积分 ≥ 150 |
| 阶段五 | 完成阶段四至少1个场景 + 训练积分 ≥ 200 |
| 阶段六 | 完成阶段五至少1个场景 + 训练积分 ≥ 300 |

**解锁校验逻辑：**

1. 查询用户训练积分
2. 查询用户在各阶段的完成记录
3. 同时满足积分+完成条件才解锁
4. 解锁后前端刷新场景列表
5. 解锁事件记入成就系统

## 7. 高难度关卡系统

### 7.1 关卡列表

| 关卡名称 | 难度 | 类型 | 描述 |
|---------|------|------|------|
| 加薪谈判·被动型老板 | ⭐⭐⭐⭐⭐ | 职场 | 面对消极被动、不愿表态的老板争取加薪 |
| 加薪谈判·强势型老板 | ⭐⭐⭐⭐⭐ | 职场 | 面对强势压制、否定你价值的老板争取加薪 |
| 加薪谈判·认可型老板 | ⭐⭐⭐⭐ | 职场 | 面对已经认可你的能力、但以预算为由推脱的老板，如何突破他的借口争取加薪 |
| 投诉物业 | ⭐⭐⭐⭐ | 生活 | 向物业投诉长期未解决的问题 |
| 4S店砍价 | ⭐⭐⭐⭐⭐ | 生活 | 在4S店与销售谈判车辆价格 |
| 路边摊砍价 | ⭐⭐⭐ | 生活 | 在路边摊与摊主砍价 |
| 投诉客服 | ⭐⭐⭐⭐ | 生活 | 向客服投诉产品质量问题，争取合理赔偿 |

### 7.2 解锁与通关条件

| 项目 | 条件 |
|------|------|
| 解锁条件 | 训练积分 ≥ 300 |
| 通关条件 | 单次训练获得三星评价 |
| 重试机制 | 无次数限制，但每次重试消耗1次训练次数 |
| 奖励 | 首次三星通关获得双倍训练积分 + 专属成就卡片 |

### 7.3 高难度关卡特殊设计

- **多分支结局**：根据关键选择节点，产生2-3种不同的对话走向
- **突发事件**：对话中随机插入意外情况（如对方突然发怒/提出额外要求）
- **时间压力**：部分关卡设置回复倒计时，模拟真实压力
- **NVC强制应用**：特定轮次要求使用NVC四步法，否则扣分
