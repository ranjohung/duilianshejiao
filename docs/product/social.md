# 变美联动·朋友圈·社交裂变

> 来源：PRD v2.0 第14-15章、第18章

## 14. 变美联动模块

### 14.1 联动概述

与"悦己颜值社"App进行深度联动，打通社交训练与形象提升的闭环。

### 14.2 联动内容

| 联动项 | 提供方 | 接收方 | 说明 |
|--------|-------|--------|------|
| 形象提升建议 | 悦己颜值社 | 「对练社交」 | 基于用户形象数据推荐社交自信提升方向 |
| 社交自信与形象关系 | 「对练社交」 | 悦己颜值社 | 用户的社交训练进展和自信变化 |
| 形象+社交综合建议 | 双方 | 用户 | 交叉分析给出综合提升方案 |

### 14.3 联动入口

| 入口位置 | 展示形式 | 点击行为 |
|---------|---------|---------|
| 首页卡片 | "形象加分，社交更自信"卡片 | 跳转悦己颜值社/展示建议 |
| 我的页 | 变美联动设置项 | 管理联动权限和数据共享 |

### 14.4 数据共享

- **前提**：用户明确授权后才共享
- **共享内容**：训练进度、能力变化趋势、社交自信评分
- **不共享**：训练对话内容、情绪日记详情、个人身份信息
- **撤回**：用户可随时在"我的页→隐私设置"中关闭联动

## 15. 朋友圈动态系统

### 15.1 功能描述

AI教练每日自动生成1条朋友圈动态，模拟真实社交中朋友分享日常的体验，增强教练的存在感和用户与教练之间的情感连接。

### 15.2 动态生成规则

| 项目 | 规则 |
|------|------|
| 生成频率 | 每个教练每日1条 |
| 生成时间 | 随机在8:00-22:00之间 |
| 动态类型 | 生活分享(40%)、心情表达(30%)、训练鼓励(30%) |
| 评论功能 | 用户可评论，教练会回复（延迟5-30分钟模拟真实感） |

### 15.3 动态内容示例

**生活分享类：**
> "今天去了一家新开的咖啡店，拿铁上的拉花是一只小猫🐱 好可爱！你们喜欢喝什么咖啡呀？"

**心情表达类：**
> "外面下雨了，突然觉得下雨天特别适合思考。你们下雨天喜欢做什么呢？"

**训练鼓励类：**
> "看到好多小伙伴今天都完成了训练，真棒！💪 记得社交就像肌肉，越练越强哦～"

### 15.4 技术实现

- 定时任务(node-cron)每日为每个教练生成1条朋友圈
- 朋友圈内容由LLM生成，基于教练人格+当前情绪+用户相关记忆
- 评论回复由LLM实时生成
- 朋友圈列表按时间倒序，保留最近30条

## 18. 社交裂变与好友邀请

### 18.1 邀请好友奖励

| 行为 | 邀请人奖励 | 被邀请人奖励 |
|------|-----------|------------|
| 好友注册 | 各类道具×1 + 10积分 | 体验卡(1次DeepSeek对话) |
| 好友完成首次训练 | 20积分 | - |
| 好友购买会员 | 50积分 | - |

### 18.2 分享学习卡片

- 训练结束后学习卡片可分享到微信/微博等社交平台
- 分享为精美图片格式（包含卡片内容+App二维码）
- 每日分享上限3次获取积分，超出仍可分享但不获得积分

### 18.3 成就卡片分享

- 达成里程碑后自动生成成就卡片
- 卡片为可分享的精美图片
- 分享无积分奖励，但提升产品曝光

### 18.4 组队训练

| 项目 | 规格 |
|------|------|
| 人数 | 2-4人 |
| 方式 | 好友一起挑战同一场景 |
| 机制 | 每人轮流与AI对话，其他人在旁观战+评论 |
| 评分 | 独立评分 + 团队协作加分 |
| 奖励 | 团队额外积分奖励(每人+10积分) |

**组队流程：**

1. 用户创建房间/加入好友房间
2. 等待所有成员就绪
3. 选择场景开始训练
4. 轮流对话，旁观者可发送表情/简短评论（评论内容经过违禁词过滤，并记录到训练日志中用于纠纷处理）
5. 训练结束各自获得评分+团队协作加分
