# 产品概述

> 来源：PRD v2.0 第1-2章

## 1. 文档概述与产品定位

### 1.1 产品定位

「对练社交」是一款 **AI驱动的对话社交训练App**，通过AI教练实时互动，结合CBT（认知行为疗法）、依恋理论、NVC（非暴力沟通）等心理学框架，为用户提供**结构化的社交技能训练**与**可量化的沟通能力提升**。

**定位于"社交技能训练平台"，不是"虚拟伴侣"或"情感陪伴"产品。**

### 1.2 一句话价值主张

> **"先在AI教练这里练一遍，再去面对真实世界。"**

### 1.3 核心差异化（vs 传统AI聊天）

| 维度 | 传统AI聊天 | 「对练社交」 |
|------|-----------|---------|
| **训练目标** | 无明确目标，泛泛而谈 | 每次对话有明确的社交能力训练目标 |
| **心理学框架** | 无 | CBT、依恋理论、NVC等心理学框架驱动 |
| **训练结构** | 自由聊天，无结构 | 场景化训练，有开始-过程-评估闭环 |
| **反馈机制** | 简单回复 | 多维度即时反馈+训练后评估报告 |
| **成长闭环** | 无成长追踪 | 能力雷达图+进步曲线+里程碑体系 |
| **形象联动** | 无 | 与"悦己颜值社"联动，社交自信与形象提升 |
| **合规设计** | 无实名认证/年龄分层/防沉迷/心理响应 | 实名认证 + 年龄分层 + 防沉迷 + 心理健康三级响应 |

### 1.4 技术目标

| 指标 | MVP目标 | 正式版目标 |
|------|---------|-----------|
| AI推理响应时间 | < 3秒 | < 2秒 |
| 教练渲染 | 2D动态立绘(Spine) | 3D渲染 ≥ 30fps |
| 语音延迟 | - | < 500ms |
| DAU | 1万 | 10万 |
| 并发训练 | 500 | 5000 |

## 2. 用户画像

### 2.1 核心用户

- **年龄段**：18-35岁都市年轻人
- **性别分布**：女性占比65-70%
- **地域**：一二线城市为主
- **消费能力**：中等以上，愿意为自我提升付费

### 2.2 四类典型用户

#### 用户A：社交焦虑人群（22-30岁）

- **特征**：在社交场合感到紧张不安，回避社交互动，担心被评价
- **核心需求**：在安全环境中练习社交，逐步建立信心
- **典型场景**：不敢参加聚会、害怕与陌生人交谈、在公共场合紧张
- **使用动机**：低成本试错，无真实社交压力

#### 用户B：职场新人（22-28岁）

- **特征**：初入职场，缺乏职场沟通经验，不知如何与上级/同事沟通
- **核心需求**：掌握职场沟通技巧，提升职业形象
- **典型场景**：面试紧张、汇报磕巴、不懂得体表达
- **使用动机**：职业发展刚需，希望快速提升

#### 用户C：沟通能力提升者（25-35岁）

- **特征**：有社交基础但希望精进，追求高阶沟通技巧
- **核心需求**：系统化提升沟通能力，学习特定技巧
- **典型场景**：想学谈判技巧、希望提升共情能力、改善亲密关系沟通
- **使用动机**：自我成长需求，愿意投入时间系统学习

#### 用户D：高难度场景需求者（22-40岁）

- **特征**：面临特定高难度社交场景，需要针对性训练
- **核心需求**：针对特定场景的实战演练
- **典型场景**：加薪谈判、投诉维权、砍价
- **使用动机**：迫切的现实需求，目标明确
