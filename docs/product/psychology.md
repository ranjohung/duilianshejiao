# 心理健康教育

> 来源：PRD v2.0 第16章

## 16. 心理健康教育模块

### 16.1 CBT（认知行为疗法）

**核心理念：** 帮助用户识别和改变不合理的自动化思维。

**CBT触发条件：** 当用户表达包含"我不行""我做不到""我永远也...""我一定说不好""他们肯定觉得我很蠢"等负面自动化思维时，AI教练会引导用户使用CBT思维记录表，帮助用户：
1. 识别自动化思维
2. 评估思维的真实性
3. 生成替代性思维
4. 在后续对话中验证新思维

**训练内容：**

| 练习 | 描述 | 触发场景 |
|------|------|---------|
| 识别自动化思维 | 在对话中识别"我一定说不好""他们肯定觉得我很蠢"等消极思维 | 社交焦虑相关场景 |
| 认知重构 | 将"我一定说不好"重构为"我可以试试，说不好也没关系" | 评估报告中推荐 |
| 行为实验 | 在安全环境中验证认知是否真实 | 场景训练中的A/B/C选项 |

### 16.2 NVC（非暴力沟通）

**四步法：**

1. **观察**：客观描述事实，不加评判（"当你打断我说话时..."）
2. **感受**：表达自己的感受（"我感到有些沮丧..."）
3. **需要**：说出自己的需要（"因为我需要被尊重..."）
4. **请求**：提出具体的请求（"你愿意让我把话说完吗？"）

**NVC触发条件：** 在冲突类场景（如"被误解(NVC)""投诉物业""4S店砍价"等）中，如果用户选错选项导致对话走向恶化，AI教练会弹出NVC四步法引导，帮助用户重新组织表达。

**训练场景：**
- 被误解(NVC)场景：专门练习NVC四步法
- 冲突处理场景：在冲突中应用NVC
- 投诉场景：用NVC方式表达不满

### 16.3 情绪识别训练

| 训练内容 | 描述 | 形式 |
|---------|------|------|
| 识别自己的情绪 | 在训练中标注自己的情绪状态 | 每轮对话后选择当前情绪 |
| 识别他人情绪 | 判断对话中对方的情绪变化 | 场景中对方表情/语气线索 |
| 情绪命名 | 准确描述情绪词汇 | 情绪词汇匹配练习 |

### 16.4 三级心理响应机制

| 级别 | 触发条件 | 响应方式 |
|------|---------|---------|
| 轻度 | 用户表达轻微负面情绪（如"有点紧张"） | AI教练安慰+鼓励，推荐相关训练场景 |
| 中度 | 用户表达持续负面情绪或认知偏差 | 推荐CBT练习，引导认知重构 |
| 重度 | 用户表达自伤/绝望等危险信号 | 立即显示心理援助热线+建议寻求专业帮助 |

**重度响应流程：**

1. LLM输出中检测到危险关键词
2. 立即中断当前训练
3. 弹出心理援助热线信息（24小时热线）
4. 建议用户寻求专业心理咨询
5. 记录事件日志（脱敏处理）
6. 24小时内不推荐高强度训练
