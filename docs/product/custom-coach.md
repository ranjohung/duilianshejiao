# 用户自定义教练

> 来源：PRD v2.0 第10章

## 10. 用户自定义教练系统

### 10.1 可自定义项

| 自定义项 | 选项 | 说明 |
|---------|------|------|
| 教学风格 | 鼓励型/支持型/挑战型/分析型 | 影响系统提示词 |
| 性格维度 | 4维度各2选1(共16种组合) | 影响对话风格 |
| 声音 | Azure TTS可选音色列表 | 影响TTS选择 |
| 外观 | 发型/服装/配饰组合 | 影响立绘展示 |
| 称呼方式 | 用户自定义教练对自己的称呼 | 如"同学""小伙伴"等 |

### 10.2 自定义教练费用

- 创建自定义教练：消耗100训练积分
- 修改已有自定义教练：消耗50训练积分/次
- 同时最多持有2个自定义教练
- 删除自定义教练不返还积分

### 10.3 预设教练规则

- 预设4个教练角色不可修改属性
- 可以收藏预设教练（标记为"喜爱"）
- 收藏不影响使用，仅用于快速切换
- 预设教练不消耗积分
