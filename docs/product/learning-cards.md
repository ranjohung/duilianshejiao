# 时空穿梭与学习卡片

> 来源：PRD v2.0 第13章

## 13. 时空穿梭与学习卡片系统

### 13.1 时空穿梭

**功能描述：** 在训练对话中，用户可以使用"时空穿梭券"回到关键分支点，重新选择不同的对话选项。

**使用流程：**

1. 训练中，每轮选择后显示"穿梭"按钮
2. 点击"穿梭"→消耗1张时空穿梭券
3. 画面出现时空倒流动效
4. 回到该轮选择前，重新展示A/B/C选项
5. 用户重新选择，教练基于新选择继续对话
6. 原选择和新选择的对比展示（可选）

**限制规则：**

- 每次训练最多使用2张
- 只能回退到当前轮次，不能跨轮回退
- 使用穿梭券不影响训练评分（鼓励试错学习）
- 穿梭后教练会特别点评两个选择的差异

**时空穿梭券获取方式：**

| 获取渠道 | 数量 | 说明 |
|---------|------|------|
| 每日签到 | 第3、5、7天各获得1张 | 签到奖励 |
| 周卡会员 | 每周3张 | 会员权益 |
| 月卡会员 | 每月10张 | 会员权益 |
| 年卡会员 | 每月15张 | 会员权益 |
| 邀请好友 | 各获得1张 | 社交裂变 |

### 13.2 学习卡片

**自动生成规则：**

- 每次训练结束后自动生成1张学习卡片
- 卡片内容由LLM基于训练过程提取

**卡片内容结构：**

```
┌──────────────────────────────────┐
│  📋 学习卡片                      │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  标题：如何在面试中展示自信         │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  关键知识点：                      │
│  • 开场时保持眼神接触             │
│  • 用具体事例代替空泛描述          │
│  • 回答前先停顿1-2秒组织语言      │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  你的表现：                        │
│  ✅ 能用具体例子支撑观点           │
│  ⬆️ 回答前可以多停顿思考           │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  改进建议：                        │
│  尝试使用STAR法则组织回答          │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  [收藏]  [分享]                   │
└──────────────────────────────────┘
```

**卡片操作：**

- **收藏**：加入我的学习卡片集，方便复习
- **分享**：生成精美图片，分享到社交平台
- **复习**：重新查看卡片内容，获得少量才艺经验(10经验)
