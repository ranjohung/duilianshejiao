# 合规实施

> 来源：PRD v2.0 第19章

## 19. 合规实施目录

### 19.1 实名认证

| 项目 | 规格 |
|------|------|
| 认证方式 | uni一键登录（首选，¥0.02-0.03元/次）+ 阿里云实人认证（¥1元/次）+ uni实人认证（备选，¥0.85元/次起），系统自动识别优先用便宜的 |
| 触发时机 | 注册即触发实名认证 |
| 认证等级 | Level1: 手机号验证(基础功能) / Level2: 身份证+人脸(训练功能) |
| 数据存储 | 认证结果存储，原始身份信息不存储(脱敏处理) |
| 重新认证 | 可在我的页重新发起实人认证 |

### 19.2 年龄分层

| 年龄段 | 功能权限 | 内容限制 |
|--------|---------|---------|
| <14岁 | 禁止注册使用 | - |
| 14-18岁 | 可使用基础训练功能 | 禁止高难度关卡(投诉/砍价/谈判)、限制每日训练1小时、内容过滤(去除成人社交场景) |
| 18+ | 全功能开放 | 无额外限制 |

### 19.3 防沉迷系统

**分阶段提醒与强制休息规则：**

| 连续训练时长 | 响应措施 |
|-------------|---------|
| 30分钟 | 弹窗提醒"已训练30分钟，休息一下吧" |
| 1小时 | 弹窗提醒"已训练1小时，建议起身活动" |
| 1.5小时 | 弹窗提醒"已训练1.5小时，建议休息15分钟" |
| 2小时 | **强制锁定训练功能15分钟**，弹窗"训练时间已到，休息一下吧，15分钟后可以继续" |

**其他规则：**

| 规则 | 规格 |
|------|------|
| 每日训练上限(未成年人) | 1小时 |
| 每日训练上限(成年人) | 4小时（用户可手动调整） |
| 锁定期间 | 可查看学习卡片、成长数据，不可开始新训练 |
| 未成年人保护 | 14-18岁每日训练不超过1小时，禁止高难度关卡 |

### 19.4 内容安全

| 项目 | 规格 |
|------|------|
| LLM输出过滤 | 违禁词库过滤 + 越界检查 |
| 过滤范围 | 政治敏感、色情暴力、歧视言论、自伤引导 |
| 过滤方式 | 关键词匹配(前置) + LLM自检(后置) |
| 违规处理 | 过滤后替换为安全内容 + 记录日志 |
| 用户举报 | 对话中可举报不当内容 |

### 19.5 隐私保护

| 项目 | 规格 |
|------|------|
| 数据加密 | 传输层HTTPS、存储层AES-256加密敏感字段 |
| 数据删除 | 用户可在"我的页→隐私设置"申请删除全部数据 |
| 数据导出 | 用户可导出个人数据(JSON格式)。导出数据包含：用户资料、训练记录、情绪日记、学习卡片，不包含对话原始内容（仅统计信息） |
| 隐私政策 | 注册时展示并要求确认 |
| 第三方共享 | 仅在用户明确授权后共享(如变美联动) |
| 对话数据 | 训练对话用于改善AI体验，但做脱敏处理 |

### 19.6 未成年人保护

| 项目 | 规格 |
|------|------|
| 内容绿色化 | 未成年人场景内容全部经过审核，确保正向引导 |
| 家长监控 | 14-18岁用户可绑定家长手机号，家长接收训练时长报告 |
| 消费保护 | 未成年人单笔消费上限50元，月累计上限200元 |
| 社交保护 | 未成年人不可使用组队训练中的陌生人匹配 |
