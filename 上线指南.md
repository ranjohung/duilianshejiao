## 📋 你需要做的事（按顺序）

### 第一步：申请免费数据库

推荐使用 **PlanetScale**（免费，无需信用卡）：

1. 打开网站：https://planetscale.com/
2. 点击 **Sign Up** → 使用 GitHub 账号登录（没有GitHub就注册一个）
3. 登录后点击 **Create database** 按钮
4. 填写信息：
   - **Name**: `duilian_social`（复制粘贴这个名字）
   - **Region**: 选择离你近的区域，推荐 Asia Pacific (Tokyo)
5. 点击 **Create database** 等待创建（约1分钟）
6. 创建完成后，点击 **Connect** 按钮
7. 在连接方式中选择 **Node.js**
8. 复制以下信息（后面要用）：
   - **Host**: xxx.us-east-2.psdb.cloud
   - **Username**: xxx
   - **Password**: xxx
   - **Database**: duilian_social

---

### 第二步：在 Vercel 配置环境变量

1. 打开：https://vercel.com/junyingera/duilianshejiao/settings/environment-variables
2. 点击 **Add** 按钮添加变量
3. 需要添加的变量（一个一个添加）：

| Key（复制这个） | Value（粘贴你从PlanetScale复制的值） |
|----------------|-------------------------------------|
| `DB_HOST` | 你的数据库 host |
| `DB_USERNAME` | 你的数据库 username |
| `DB_PASSWORD` | 你的数据库 password |
| `DB_NAME` | `duilian_social` |
| `DB_PORT` | `3306` |
| `DEEPSEEK_API_KEY` | `sk-96d7c2f77e2f47beacc75ebd17172dd1` |
| `JWT_SECRET` | `duilian-social-2026-secret-key` |

4. 每个变量都要勾选 **Sensitive**（保持蓝色）
5. 每个变量的 Environments 都选择 **Production and Preview**
6. 全部添加完后点击 **Save**

---

### 第三步：初始化数据库表

1. 打开浏览器，访问：https://duilianshejiao.vercel.app/health
   - 如果显示 "OK"，说明后端服务正常
2. 打开另一个标签页，访问：
   ```
   https://duilianshejiao.vercel.app/api/init-db
   ```
   - 如果显示 `{"message":"数据库初始化成功"}`，说明成功！
   - 如果显示错误，检查第二步的数据库信息是否填对

## ❓ 常见问题

**Q: 网站打不开怎么办？**
A: 检查网络连接，尝试刷新页面。如果持续打不开，可能是Vercel部署还在进行中，等待几分钟再试。

**Q: 登录失败怎么办？**
A: 检查数据库是否配置正确，确认第三步的数据库初始化是否成功。

**Q: AI对话没反应怎么办？**
A: 检查 `DEEPSEEK_API_KEY` 是否正确配置，确认API Key还有余额。

**Q: 需要花钱吗？**
A: PlanetScale免费版足够使用，DeepSeek API有免费额度，Vercel免费版足够。
